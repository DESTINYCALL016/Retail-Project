Project Title:
Retail Intelligence through SQL: Customer Loyalty and Transaction Insights

Author: Ayush Chaudhary
Date: April 2025
Project Type: Internship Final Report Presentation
Organization: Scaletrix.ai

Purpose:
This PowerPoint presentation summarizes a comprehensive retail analytics project conducted during an internship program. The goal was to derive actionable insights from retail transaction data using SQL and present it to a non-technical audience in a structured, clear, and visual format.
